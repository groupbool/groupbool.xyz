tritones

Here at groupbool, every thursday is tritone thursday where we pick three songs that share a common theme and write a short description about the overarching narrative.

We’re looking for more authors. If you have any suggestions or want to write a tritone for groupbool, please reach out to us at requests@groupbool.xyz
